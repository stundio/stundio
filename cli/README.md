### All files in this directory are used to **test the data extractor(s)**.

These files were generated by sending a request to the following URL:

```bash
curl "https://pikcrvt.edupage.org/timetable/server/regulartt.js?__func=regularttGetData"
```

### Note
The response from this URL is a JavaScript file that contains timetable data in JSON format wrapped in a function call.
Each test file corresponds to a snapshot of the data at the time of the request and can be used to verify the correctness of the data extractor.
